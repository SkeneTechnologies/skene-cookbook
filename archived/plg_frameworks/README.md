# Archived Skills

This directory contains the original format PLG framework skills that were replaced by enhanced plgf_* versions.

## Why Archived?

These skills existed in two versions:
1. Original format (e.g., `activation-metrics`) - Simple, no tools
2. Enhanced format (e.g., `plgf_activation_metrics`) - With tools, schemas, metrics

The enhanced versions provide:
- Tool integrations
- Complete input/output schemas
- Metrics and benchmarks
- Better structured metadata

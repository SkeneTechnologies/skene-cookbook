## Usage

```typescript
import { SkillRegistry } from './src/headless/SkillRegistry';

const registry = new SkillRegistry('./skills-library');
await registry.indexAll();

// Load any skill by domain/name
const skill = await registry.loadSkill('security/semgrep-rule-creator');
const scientific = await registry.loadSkill('scientific/biopython');
const plg = await registry.loadSkill('plg_frameworks/activation-metrics');
```

## The Boyce Framework

The Boyce PLG framework focuses on:

1. **First Impact** — Delivering value before signup
2. **Acquisition Loops** — Self-sustaining growth mechanisms
3. **Usage Retention** — Product-driven engagement
4. **Expansion Revenue** — Natural upgrade paths
5. **Product-Led Sales** — Sales informed by product usage

Each skill in this directory implements a specific aspect of the framework.

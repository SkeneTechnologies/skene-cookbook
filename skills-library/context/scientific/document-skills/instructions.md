# Document Skills

This skill provides tools for scientific document processing, including:

- **PDF Processing**: Extract text, tables, and figures from scientific papers
- **LaTeX Generation**: Create properly formatted LaTeX documents for publications
- **Document Conversion**: Convert between formats (PDF, DOCX, Markdown, LaTeX)
- **Citation Management**: Format and manage bibliographic references
- **Figure Extraction**: Extract and process figures from research papers

## When to Use

- Processing scientific papers and extracting structured data
- Creating publication-ready documents
- Converting between document formats
- Managing references and citations

# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for all contributors, regardless of background or identity.

## Standards

✅ **Do:**

- Be respectful and constructive
- Focus on skill quality and security
- Help others learn and improve
- Give credit where due

❌ **Don't:**

- Submit malicious skills
- Bypass security requirements
- Copy proprietary workflows without permission
- Harass or discriminate

## Enforcement

Violations will result in:

1. Warning
2. Temporary ban
3. Permanent ban

Report issues to: security@skene.ai

---

This Code of Conduct applies to all project spaces, including GitHub repositories, discussions, issues, pull requests, and any other community venues.

By participating in this project, you agree to abide by this Code of Conduct.

## Quick Start

### 1. Install

```bash
npm install @skene/skills-directory
```

### 2. Activate All Skills

```bash
npx skills-directory install --target all
```

This installs 800+ skills across all domains into your Claude/Cursor environment.

### 3. Browse Skills by Domain

```bash
# See all RevOps skills
npx skills-directory list --domain revops

# See all Customer Success skills
npx skills-directory list --domain customer_success

# See all Finance skills
npx skills-directory list --domain finops
```

### 4. Build Your First Chain

Start with a proven recipe from [SKILL_CHAINS.md](./SKILL_CHAINS.md), or:

1. **Pick a use case** â€” What workflow do you want to automate?
2. **Select skills** â€” Browse the domain, pick 3-5 skills
3. **Design the chain** â€” Map exit states to next skills
4. **Test & refine** â€” Start with sample data, iterate
5. **Deploy** â€” Connect to your tools, go live

### 5. Measure & Scale

- Track time saved vs manual process
- Monitor business metrics (conversion, churn, etc.)
- Expand successful chains with more skills
- Build new chains for adjacent workflows

# .ai Directory

This directory contains AI agent configuration and internal documentation.

## Structure

- `AGENTS.md` (repository root) - Primary AI agent instructions
- `.ai/internal/` - Internal documentation (gitignored, not indexed by AI agents)

## Purpose

- `AGENTS.md` provides build commands, testing workflows, and conventions for AI coding agents
- `.ai/internal/` stores maintainer-only notes that shouldn't be indexed by AI agents

## See Also

- `.cursorignore` - Specifies files/directories to exclude from Cursor AI indexing
